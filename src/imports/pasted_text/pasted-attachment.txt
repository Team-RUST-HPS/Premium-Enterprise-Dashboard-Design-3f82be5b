Figma AI Prompt — High-Speed AI Mapping Window

Design a next-generation schema mapping window for an enterprise data transformation platform.

The purpose of this interface is to allow users to define and manage extremely large mappings (up to 100,000 field mappings) in a very fast, visually clean, and relaxing environment.

The UI should prioritize speed, clarity, and automation, allowing users to confirm or adjust mappings quickly rather than manually drawing connections.

The visual quality should match modern enterprise tools similar to dashboards built by Stripe and Vercel.
